# ❤️ Heart Disease Prediction using Machine Learning

A Streamlit web application that predicts whether a patient is at risk of heart disease using a Logistic Regression model trained on the UCI Heart Disease Dataset.

---

## Features

- Interactive Streamlit interface
- Logistic Regression classifier
- Prediction confidence using probabilities
- Patient summary
- Risk level visualization
- Responsive two-column layout

---

## Dataset

UCI Heart Disease Dataset

Features include:

- Age
- Sex
- Chest Pain Type
- Blood Pressure
- Cholesterol
- Fasting Blood Sugar
- ECG Results
- Maximum Heart Rate
- Exercise Induced Angina
- Oldpeak
- ST Slope
- Major Vessels
- Thalassemia
